Escaped CSS Selector Test

Each box below should have its specified background color if the selector is working.

Test 1: .simple-class → Should be GREEN
Test 2: .lg\:w-1024 → Should be BLUE (width 1024px)
Class attribute: lg:w-1024
Test 3: .w-\[500px\] → Should be PURPLE (width 500px)
Class attribute: w-[500px]
Test 4: .lg\:w-\[800px\] → Should be ORANGE (width 800px)
Class attribute: lg:w-[800px]
Test 5: .mq\:test (inside @media) → Should be RED
Class attribute: mq:test

Expected Results:

  • Test 1 (simple-class): GREEN background
  • Test 2 (lg:w-1024): BLUE background, 1024px wide
  • Test 3 (w-[500px]): PURPLE background, 500px wide
  • Test 4 (lg:w-[800px]): ORANGE background, 800px wide
  • Test 5 (mq:test): RED background

If any box is missing its color/width, that selector type is not being parsed correctly.
